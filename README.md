# NodeJS3
NodeJS part 3

faire un git clone de ce git (https://github.com/Corentin91560/node-app.git)
se placer dans le dossier 
## installation
``` bash
$ npm install
$ DEBUG="node-app:server" npm start
```
## envoie sur heroku
```bash
$ heroku login 
$ heroku create 
$ git push heroku master
```
## lancer heroku sur le navigateur
```bash
$ heroku open
```
